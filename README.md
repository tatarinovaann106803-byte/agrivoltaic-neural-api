# Agrivoltaic Neural API

API сервис для расчёта эффективности агривольтаических систем (совместное размещение солнечных панелей и сельскохозяйственного производства).

Поддерживаемые сектора: растениеводство, аквакультура, лесное хозяйство.

## Возможности

- Автоматическое получение солнечной радиации и температуры из NASA POWER API
- Автоматическая подстановка цен на продукцию из базы Росстата
- Автоматическая подстановка базовой урожайности из FAO
- AI-предсказание изменения продуктивности (нейросеть или Random Forest)
- Расчёт энергетики
- Экономический расчёт (CAPEX, OPEX, срок окупаемости)

### Папка `.github/workflows/`

| Файл | Назначение |
|------|------------|
| `update-data.yml` | Настройка автоматического еженедельного обновления данных и переобучения моделей через GitHub Actions |

### Папка `data/` (статические базы данных)

| Файл | Назначение |
|------|------------|
| `crop_prices.csv` | Цены на сельскохозяйственные культуры по регионам России (данные Росстата). Используется для автоматической подстановки цены при расчёте. |
| `crops_yield_faostat.csv` | Базовая урожайность культур по странам мира (данные FAO). Используется для автоматической подстановки урожайности. |
| `aquaculture_production.csv` | Объёмы производства аквакультуры по странам и видам рыб (данные FAO FishStatJ). |
| `aquaculture_value.csv` | Стоимость продукции аквакультуры по странам и видам рыб (данные FAO FishStatJ). Используется для автоматической подстановки цены рыбы. |
| `forestry_production.csv` | Объёмы производства лесной продукции по странам (данные FAO). |
| `forestry_trade.csv` | Стоимость экспорта/импорта лесной продукции по странам (данные FAO). Используется для автоматической подстановки цены древесины. |

### Папка `dataset/` (данные для обучения моделей)

| Файл | Назначение |
|------|------------|
| `raw_dataset.csv` | Сырые данные, собранные из Google Sheets. |
| `sector_data/crop_farming.csv` | Данные по АВ-проектам в растениеводстве. Используются для обучения AI-модели предсказания изменения урожайности. |
| `sector_data/aquaculture.csv` | Данные по АВ-проектам в аквакультуре. Используются для обучения AI-модели предсказания продуктивности. |
| `sector_data/forestry.csv` | Данные по АВ-проектам в лесном хозяйстве. Используются для обучения AI-модели предсказания прироста древесины. |

### Корневые файлы

| Файл | Назначение |
|------|------------|
| `app.py` | Основной файл FastAPI сервера. Содержит API эндпоинты, логику расчёта энергии, экономики, а также функции подстановки цен и урожайности из баз данных. |
| `train_advanced_models.py` | Обучение Random Forest моделей для трёх секторов. Создаёт файлы `model_*.pkl` и `scaler_*.pkl` в папке `models/`. |
| `train_neural_models.py` | Обучение нейросетевых моделей (MLPRegressor) для трёх секторов. Создаёт файлы `neural_model_*.pkl` и `neural_scaler_*.pkl`. |
| `smart_train.py` | Умное обучение: автоматически выбирает модель (нейросеть или Random Forest) в зависимости от количества доступных данных. |
| `predict_neural.py` | Класс для предсказаний. Автоматически выбирает лучшую доступную модель (нейросеть → Random Forest → fallback) и возвращает результат с информацией об использованной модели. |
| `auto_train.py` | Скрипт автоматического переобучения. Проверяет, изменились ли данные (по хешу), и запускает обучение при необходимости. |
| `download_data.py` | Скачивание данных из Google Sheets (три таблицы) и объединение их в `dataset/raw_dataset.csv`. |
| `prepare_dataset_advanced.py` | Преобразование `raw_dataset.csv` в секторные файлы (`crop_farming.csv`, `aquaculture.csv`, `forestry.csv`). |
| `requirements.txt` | Список Python-зависимостей для установки через pip. |
| `runtime.txt` | Фиксация версии Python (3.11). Нужен для Render. |
| `Procfile` | Инструкция для Render: какая команда запускает сервер. |
| `README.md` | Инструкция по установке, настройке и использованию сервиса. |

## Быстрый старт

Выполните команды по порядку:

```bash
# 1. Клонирование репозитория
git clone https://github.com/tatarinovaann106803-byte/agrivoltaic-neural-api.git
cd agrivoltaic-neural-api

# 2. Создание виртуального окружения
python -m venv venv
source venv/bin/activate        # для Mac/Linux
venv\Scripts\activate           # для Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Скачивание и подготовка данных
python download_data.py
python prepare_dataset_advanced.py

# 5. Обучение моделей
python train_advanced_models.py
python smart_train.py

# 6. Запуск сервера
uvicorn app:app --host 0.0.0.0 --port 8000 --reload
